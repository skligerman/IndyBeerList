About Indy Beer List

I love making lists, and I love Indianapolis. Recently, when a friend of mine moved from Chicago to Indy, he was concerned with Indy's lack of quality local beers. So here's my way of showing off what's available, and what's coming soon.

How did I build it?

This project had an incredibly circuitous route. As a front-end designer/dev, I wanted to learn a lot more about open source and the latest web tools. First, I built the site in Ember, getting insight from Code School's fantastic learning platform, then built the site in BackBoneJS, but didn't enjoy building in it. Finally, I built the site in AngularJS because it was the most fun to learn.

The site is currently hosted using Div Shot, because they specialize in static apps and have an easy-to-use command-line deployment process. (During the early part of the process I spent many hours using the command-line and became a lot more comfortable with it). The data is stored in a JSON file, but I plan to move it into a #NoBackend service like Firebase. As mentioned earlier, the dynamic content is written in AngularJS, and the avatars are using Cloudinary's Twitter API. So, after months of reading blogs, tutorials, and watching Dribbble feeds, I (finally) decided to push the project out and iterate from there.
